## قرارات معمارية أساسية

```
شخص (Person)
 ├── النوع: زائر | مسجّل A | مسجّل B
 ├── الوصف الكامل (حسب النوع — انظر 03)
 ├── محاضر مرتبطة
 ├── أسلحة مرتبطة
 └── سجل تدقيق (Audit)
```

## مصطلحات

| المصطلح | التعريف |
|---------|---------|
| **مسجّل** | أي شخص في قاعدة البيانات (يشمل الزائر) |
| **زائر** | سجل مؤقت من حادث/بلاغ — بيانات محدودة |
| **مسجّل A** | سجل دائم — أولوية عالية — ملف كامل |
| **مسجّل B** | سجل دائم — متابعة/مراقبة — ملف كامل |
| **محضر** | تقرير حادث يربط مكاناً وزماناً وأسلوب جريمة بأشخاص |
